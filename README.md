## Shop xtarantulz
Проект - шабон магазина делался для портфолио, чтобы показать логику Yii2

## Установка
1. composer update
2. php init
3. создайте базу данных и внесите доступы в \common\config\main-local.php
4. php yii migrate или используйте бекап shop.sql в корне
